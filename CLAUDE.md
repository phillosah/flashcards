# Project Instructions

## Version Bumping

**Bump the version on every edit** — no exceptions.

- Version string is in `web/index.html` (search for `v1.x.x` near the header title)
- Follow semver: patch for fixes/style tweaks, minor for new features
- Update `CHANGELOG.md` with a brief entry for the new version
- Commit message should include the new version number
